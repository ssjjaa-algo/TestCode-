# BDD 스타일로 작성하기

- TDD에서 파생된 개발 방법
- 함수 단위 테스트보다 **시나리오에 기반한 TC**에 집중

## Given / When / Then

- Given : 시나리오 진행에 필요한 모든 준비 과정(객체, 값, 상태 등)
- When : 시나리오 행동 진행
- Then : 시나리오에 대한 결과, 검증
